# 📖 Leitura Missa — App de Liturgia Diária

Uma aplicação web moderna para acompanhar a **Liturgia Diária da Igreja Católica**, incluindo leituras da missa, 
evangelho do dia de forma simples, rápida e responsiva.

## ✨ Demonstração

🔗 Acesse o projeto: https://leitura-missa.vercel.app/

## 📌 Sobre o projeto

O **Leitura Missa** é um aplicativo web desenvolvido com o objetivo de facilitar o acesso às leituras diárias da missa, permitindo que os usuários acompanhem:

- 📅 Evangelho do dia
- 📖 Primeira e segunda leitura
- 🎶 Salmo responsorial

O foco do projeto é oferecer uma experiência limpa, rápida e acessível em qualquer dispositivo.

## 🚀 Funcionalidades

- 📆 Leitura diária automática do dia, com salvamento no localStorage 
- 📖 Visualização organizada das leituras da missa  
- 🔎 Navegação por dias anteriores e futuros  
- 📱 Interface responsiva (mobile-first)  
- ⚡ Carregamento rápido com otimização para web  
- 🌙 Layout limpo e focado na leitura

## 🧱 Tecnologias utilizadas

- 📡 API de Liturgia / dados estruturados
- ☁️ Deploy na Vercel
- 🧠 HTML, CSS e JavaScript
